== Description ==

By using this smart plugin, allows you to add Custom HTML to bottom and top of footer in your wordpress Website. No need to edit theme!

You can add custom HTML easily from admin side,there is two text editor to add HTML text for footer top and footer bottom section.  

**Plugin Features**

*Quick to set up
*Simple to insert scripts
*Insert HTML to footer
*Pre-footer HTML
*Showing to all pages
*Add custom CSS across themes


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After active plugin go to Setting->Adding Content to the Footer.
